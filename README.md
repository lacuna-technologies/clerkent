## Development

Run in development mode for your preferred browser:

```bash
npm run dev:chrome
npm run dev:firefox
npm run dev:opera
```

Load it as an unpacked extension

Hot reloading should work

### Tests

There are some tests, mostly snapshot tests.

```bash
npm run test
```
